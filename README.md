# Demoblaze Playwright Automation Framework

A complete **Playwright + TypeScript** automation framework built for testing the Demoblaze application.  
This project follows **Page Object Model (POM)**, includes **Allure Reporting**, and is ready for **CI/CD integration**.

---

## 🚀 Features

- ✔ Playwright with TypeScript  
- ✔ Page Object Model (POM)  
- ✔ Reusable utilities & fixtures  
- ✔ Allure reporting integration  
- ✔ Screenshots, videos & trace logs  
- ✔ Clean folder structure  
- ✔ Easy to run tests  
- ✔ CI/CD ready (GitHub Actions)

---

## 📁 Folder Structure
Demoblaze_playwright/
